# Hugging Face Learn PDF eBooks

A curated collection of **9 free machine learning courses** from [Hugging Face Learn](https://huggingface.co/learn), converted to beautifully formatted PDF eBooks for offline reading.

[![Hugging Face](https://img.shields.io/badge/🤗-Hugging%20Face-FFD21E)](https://huggingface.co)

## 🚀 Quick Start

### Download Individual Courses

```bash
# Clone with all PDFs
git clone https://github.com/artificialguybr/huggingface-learn-pdf-ebooks.git
cd huggingface-learn-pdf-ebooks

# Or download specific course
curl -O https://raw.githubusercontent.com/artificialguybr/huggingface-learn-pdf-ebooks/main/agents-course.pdf
```

## 🛠️ Technical Details

**Conversion Process:**
- Source: [Hugging Face GitHub repositories](https://github.com/huggingface)
- Format: MDX → HTML → PDF
- Tooling: Playwright + Chromium for accurate rendering
- Compression: Ghostscript optimization (78% size reduction)

**PDF Specifications:**
- Format: A4, 2cm margins
- Font: System fonts (Segoe UI, Roboto)
- Code: Monospace with syntax highlighting
- Images: Compressed while preserving readability
